# Shift-Reduce-Parsers
Phân tích cú pháp ngôn ngữ Python
### Thư viện sử dụng:
* nltk
* pyqt5

### Cách cài đặt:
    <pre>pip install -r requirements.txt</pre>
    
    
### Cách sử dụng:
    <pre>python main.py</pre>
